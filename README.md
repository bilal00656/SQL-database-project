# Gym Membership Management System (SQL)

## 📌 Overview
This project is a relational database system designed to manage gym operations including members, trainers, plans, payments, and attendance.

## 🧱 Features
- Member & Trainer Management
- Membership Plans
- Payment Tracking
- Attendance System
- Workout Plans
- SQL Joins, Views, Subqueries
- Stored Procedures & Triggers

## 🗂️ Project Structure
- schema.sql → Database & tables
- data.sql → Sample data
- queries.sql → SQL queries
- procedures.sql → Stored procedures
- triggers.sql → Triggers

## 🛠️ Technologies
- SQL Server (T-SQL)

## 📊 Concepts Used
- Normalization (1NF, 2NF, 3NF)
- Joins & Subqueries
- Views
- Transactions
- Constraints
- Stored Procedures
- Triggers

## 🚀 How to Run
1. Run `schema.sql`
2. Run `data.sql`
3. Execute queries from `queries.sql`
